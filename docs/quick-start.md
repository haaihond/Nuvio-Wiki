# ⚡ Quick Start Guide

Want to get Nuvio up and running in 5 minutes? Follow this sequence:

## 1. Install the App
- **Android:** Download the APK and install. [Full Guide](docs/installation/android-mobile.md)
- **Android TV:** Use the "Downloader" app to get the TV APK. [Full Guide](docs/installation/android-tv.md)

## 2. Add Your First Addon
Nuvio is empty by default. You need an addon to see content.
- Open Nuvio > **Settings** > **Addons**.
- Tap **Install from URL**.
- Enter a manifest link (e.g., from a community repo like Duckstreams).

## 3. (Optional but Recommended) Setup Debrid
For high-quality 4K streaming without buffering:
- Go to **Settings** > **Services**.
- Select your provider (Real-Debrid, AllDebrid, etc.).
- Follow the on-screen code to authorize. [Integration Guide](docs/integrations/debrid.md)

## 4. Setup Trakt
- Go to **Settings** > **Account**.
- Login with Trakt to sync your watchlist and progress.

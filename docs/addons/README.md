# Addons Guide

Addons are the heart of the Nuvio experience. They provide the content and functionality that make the app useful.

## Types of Addons

- **Content Providers:** These fetch catalogs and streams for movies, shows, or live TV.
- **Metadata Providers:** Enhance the UI with posters, ratings, and synopses.
- **Utility Addons:** Provide extra features like subtitles or playback sync.

## Finding Addons

Nuvio maintains a community-driven ecosystem. Popular sources include:
- Official Nuvio Addon Repository (built-in).
- Community lists (e.g., Duckstreams).

## Setup Links

To add an addon, you typically need a manifest URL.
- **Duckstreams Example:** `https://duckstreams.link/manifest.json` (Verify current links on their official site).
